How it works

The Intelligent Investor

This software is designed to make the process of analysing stocks easier for you. You need to simply type in the ticker symbol for a particular company then the software will advise whether the particular stock is worth adding to your portfolio.

Warren Buffet's mentor, Benjamin Franklin, wrote a widely regarded book called The Intelligent Investor. In this book Benjamin Franklin suggests a number of ratio anlaysis concepts to use when deciding whether to invest in a company. Our software uses these same checks to analyse your information.

The book is all about value investing and finding stocks that are undervalued. The software provides you with some key information to help you determine if a stock is at a good value.
